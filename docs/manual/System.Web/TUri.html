TUri class

TUri represents a URI. Given a URI http://joe:whatever@example.com:8080/path/to/script.php?param=value#anchor it will be decomposed as follows,

  • scheme: http
  • host: example.com
  • port: 8080
  • user: joe
  • password: whatever
  • path: /path/to/script.php
  • query: param=value
  • fragment: anchor
