Class TWizardStep

Description
Description | Descendents | Methods (details) Constants

TWizardStep class.

TWizardStep represents a wizard step. The wizard owning the step can be obtained by Wizard. To specify the type of the step, set StepType; For step title, set Title. If a step can be re-visited, set AllowReturn to true.
