TActiveHyperLinkColumn class

TActiveHyperLinkColumn contains a hyperlink for each item in the column.

This is the active counterpart to the THyperLinkColumn control. For that purpose, if sorting is allowed, the header links/buttons are replaced by active controls.

Please refer to the original documentation of the THyperLinkColumn for usage.
